Important!

HTML semantic structures are rendered obsolete if you incorrectly assign a class. For example, do not assign "Heading High" to a P tag. Conversely, do not add a "body-low-default" class to H1-H4 tags.

h1 + .heading-low

h2 + .title-high

h3 + .title-medium

h4 + .title-low
Known issues

  1. In Angular, the default paragraph style does not automatically load. Manually add a body class to the index.html page:

  2. Title-lowest is not mapped to any html tag and does not render the style correctly
  3. Code for mat-card-content doesn't follow padding rules
