Starterkit

Welcome to the Starterkit — a comprehensive resource for efficiently cloning Angular UI code repositories and obtaining Figma libraries, fully aligned with the Atmosphere Design Language System (Atmosphere DLS).

This kit is intended to support developers in rapidly establishing compliant user interfaces, while providing designers with templates featuring accurate variables, colors, and measurements.

refer to the templates and components sections in the navigation to review examples constructed in accordance with Atmosphere DLS standards.
