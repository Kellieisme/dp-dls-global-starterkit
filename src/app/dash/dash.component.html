Starterkit

Welcome to the Starterkit — a comprehensive resource for efficiently cloning Angular UI code repositories and obtaining Figma libraries, fully aligned with the Atmosphere Design Language System (DLS).

This kit is intended to support developers in rapidly establishing compliant user interfaces, while providing designers with templates featuring accurate component behaviors, variables, colors, and measurements.

Please refer to the templates and components section in the navigation to review examples constructed in accordance with Atmosphere DLS standards.
