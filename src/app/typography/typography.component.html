Important!

HTML semantic structures are rendered obsolete if you incorrectly assign a class. For example, do not assign "Heading High" to a P tag. Conversely, do not add a "body-low-default" class to H1-H4 tags.

h1 + .heading-low

h2 + .title-high

h3 + .title-medium

h4 + .title-low
